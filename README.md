# Raspador de dados do Facebook Webscraper

Extensão Chrome criada a partir do bookmarklet funcional validado.

## Instalação

1. Extraia o ZIP.
2. Abra o Chrome.
3. Acesse: chrome://extensions/
4. Ative o Modo do desenvolvedor.
5. Clique em Carregar sem compactação.
6. Selecione a pasta extraída `Raspador_de_dados_do_Facebook_Webscraper`.

## Uso

1. Abra uma publicação, foto ou Reel do Facebook.
2. Clique no ícone azul com as letras `fb`.
3. Aguarde a raspagem.
4. Os arquivos serão baixados automaticamente pelo próprio script.

## Observação

O núcleo do bookmarklet foi preservado.
